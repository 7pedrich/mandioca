# mandioca
This is a pretty cool, yet uncomplete chess engine I've made. I may or may not continue this.

This engine has almost all of the basic features it needs to include:
---
- Piece movement
- Checks
- Captures
- En Passant
- Castling (Kingside & Queenside)
- Turn to move

Though it lacks:

- Pawn promotion and underpromotion
- King-safety
- Discovery check detection
